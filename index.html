Collect all five Elements to win!

WaterFire MetalWood Earth

The order of the cards is not important.

Light is wild and can represent any element you need to win

Dark does not contribute to winning and is often used to block your opponent's slot.

You must always play your card even if you don't like the result.

Element Card

An element card can be played into any empty slot or replace any element it can defeat.

You can see the colors of the replaceable elements at the top left corner of each card.

Action Card

Move

Choose one Element card from any player’s row and move it to an empty slot in a different player’s row.

Swap

Choose one Element card from any player’s row and swap it with a different Element card from a different player’s row.

Discard 1 / Discard 2

Discard an equal number of Element cards from the table as the Discard type (shown by the arrows).
Cards can be chosen together from the same player (including yourself) or separately from different players.
